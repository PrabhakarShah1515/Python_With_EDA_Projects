# Kabaddi Data Analysis (EDA) with Python

## Project Overview
This project performs **Exploratory Data Analysis (EDA)** on Kabaddi match and player data to uncover insights and trends in the sport. Kabaddi is a popular team sport in South Asia, and this analysis aims to explore player performance, team statistics, and match outcomes to identify patterns and key factors influencing results.
